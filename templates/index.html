Syntax Guide

  • λx.x - Lambda abstraction (identity function)
  • (λx.x) y - Function application
  • + 2 3 - Arithmetic (prefix notation)
  • * 2 3 - Multiplication
  • true, false - Boolean values
  • Numbers - Automatically converted to Church numerals
